CameraPreview

@Composable
fun CameraPreview(onBind: OnBindScope.() -> Unit, modifier: Modifier = Modifier, onInitPreviewView: PreviewView.() -> Unit = {}, lifecycleOwner: LifecycleOwner = LocalLifecycleOwner.current)

カメラのPreviewを表示するための Composable 関数。 プレビューの表示にはonBindコールバック内でPreviewユースケースをbindする必要があります。

使用例

CameraPreview(
  onBind = {
    val preview = previewUseCase()
    cameraProvider.bindToLifecycle(
      lifecycleOwner,
      CameraSelector.DEFAULT_BACK_CAMERA,
      preview,
    )
  },
)

onBind

UseCaseを用意してcameraProvider.bindToLifecycleを呼び出してください。レシーバとしてOnBindScopeを受け取るため、previewViewやcameraProviderなどにアクセスできます。
